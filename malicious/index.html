CSRF ATTACK DEMONSTRATION SUITE

Explore Cross-Site Request Forgery vectors

EDUCATIONAL PURPOSE ONLY: These attacks demonstrate CSRF vulnerabilities in controlled environments. Pair this page with the vulnerable user management build and only test in lab environments you control.

Attack 1: Fetch CSRF

Uses JavaScript fetch() API to make cross-origin GET request with credentials

Attack 2: Image Tag CSRF

Creates hidden image tag to trigger GET request automatically

Attack 3: Form Submission

Auto-submits hidden form via iframe to target endpoint

Attack 4: PUT Request CSRF

Sends authenticated PUT request to change username to "TREVOR_"
